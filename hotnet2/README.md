## [hotnet2](https://github.com/raphael-group/hotnet2)
### runHotnet2.py:
### with or without mutsig
```shell
runhotnet2.py
--hotnet2 $PATH2hotnet2 \
--maf_file  $maf_file \
[ --mutsig_file $mutsig_file ] \
[ --threshold q0.01 ] \
[ -o $output_dir ]\
[ -p $thread_num ] \
[ --cna_file $Scna_file ]
```
